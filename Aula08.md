### 	Coleção List

Comando append adiciona um objeto no final da lista

```python
carros = []
carros.append("Palio")
```

Comando "remove" exclui um item da lista

```python
carros = ["Mustang", "Gol"]
carros.remove("Gol")
```

Comando pop remove o último item da lista

```python
carros = ["Golf", "Focus", "Palio"]
carros.pop
#Resultado seria Golf, Focus
```
